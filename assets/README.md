# Tray Icon Assets

This directory contains icon files for the Windows system tray application.

## Required Icons

- `tray_icon.png` - Normal state icon (16x16, 32x32, 48x48 pixels)
- `tray_icon_warning.png` - Warning state icon (health check failed)
- `tray_icon_error.png` - Error state icon (service crashed)

## Icon Requirements

- Format: PNG or ICO
- Minimum dimensions: 16x16 pixels
- Recommended sizes: 16x16, 32x32, 48x48 (for different DPI settings)
- Transparent background recommended
